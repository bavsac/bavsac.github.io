ramblr - Built this app in colobration with 4 other developers.

Last year Mountain Rescue England and Wales had to respond to over 3000 callouts in the countryside...Ramblr is an app designed to help curb those figures. With Ramblr you are able to log in, set a desired time for your ramble and add a trusted contact's phone number. If you don't alert the app that you have completed your walk, Ramblr will automatically send a text message at the end of your set time to your trusted contact with a live location of your whereabouts. Tech used: JavaScript | AWS Amplify | AWS Cognito | AWS DynamoDB | AWS Lambda | AWS CloudWatch | React Native | Expo | CSS | SMSWorks API
